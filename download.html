Which version?


Raven B+ has a few options to choose from! Some are in heavy development, for example Beta and Experimental. These may have bugs, or have slightly different modules from each other.

Experimental is recomended as it's very stable right now, and it is the least problematic.

Stable - Has the regular Raven B3 GUI and will not receive any updates.

Download the latest Stable version. Just drag and drop this into your 1.8.9 forge mod folder, launch, and play!

Find the Github for the Stable branch here!

Experimental - An experimental version with a slightly revamped base!

Download the latest Experimental branch. Just drag and drop this into your 1.8.9 forge mod folder, launch, and play!

Find the Github for the LunarKeyStrokes branch here!

Beta - An unstable branch, but has unique features.

Download the latest Beta build. Just drag and drop this into your 1.8.9 forge mod folder, launch, and play!

Find the Github for the Beta (New-Base) branch here!

Github

All versions are open on Github for you to modify. Under theGPL-3.0 license any versions you plan to make should be open source and free!

Beta

Please don't complain about crashes on Beta or Experimental. These versions are labelled as unstable and use at your own risk. Feel free to report the crash log, but just know it may crash.

Stable

Stable is recommended for people who are inexperienced with cheating, and don't want to risk anything to chance.
